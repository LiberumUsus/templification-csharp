Tag Templates

Tag Templates are templates wrapped in a tag with the 2 underscore template prefix, like: <__templateName.... After they are defined they are used like any other tag.

These templates when they are on the root level of a file can be used just like any other file based template. If they are embedded within other tags they are  ONLY available within that file.

Local/File Template Tags can have script tags, style tags, logic blocks and any other standard template definition.

Root Level Tag Templates

<__Button>...</__Button>
<__TextBox>...</__TextBox>
<__Select>...</__Select>
*This serves the case where many
templates are embedded in a single file; for instance HTML controls like checkbox, select and textbox.
