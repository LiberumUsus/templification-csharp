Slots and Defaults

Default slots are specificed with the <slot/> tag inside a template.

Additionaly the {default}variable can be inserted into a template. When it is present the first element or text from the source is inserted into this location in the template.

The default value can also be set via an attribute like any other variable value.
e.g.<TagName {default}="value"...
