Generating Classes

Classes can be generated based on preconfigured names with postfix patterns. A simple example would be border-t-2 where a border will be applied on the top of the element with a 2px width.

The following element types are currently supported with their corresponding class attribute.

Generated Classes: Simple px Value
'p':'padding': 'p[tlbrxy]':'padding-<direction>': 'm':'margin': 'm[tlbrxy]':'margin-<direction>': 'h':'height': 'w':'width': 't':'top': 'l':'left': 'r':'right': 'b':'bottom': 'underline':'text-underline-offset': 'maxw':'max-width': 'gap':'gap'
Negative Values

Add a -before any class name to generate a negative value of the same.

i.e.: -t-2generates a position from the top with a negative value.
